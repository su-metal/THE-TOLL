## エグゼクティブサマリー

THE TOLLは「運動しないとWebサイトが見れない」という強制的なフリクションを課すことで、SNS依存と運動不足という2つの問題を同時解決するユニークなプロダクトである。この独自性は、競合の単純なサイトブロッカーと明確に差別化できる。

本レポートでは、行動経済学の知見に基づき、以下を提案する:

- **推奨価格**: Pro月額 $4.99/月 (USD)、¥750/月 (JPY)
- **推奨プラン構成**: Free / Pro / Pro Annual（3ウェイ構成）
- **トライアル**: 14日間フル機能（現行7日から延長）
- **ローンチ戦略**: 最初30日間は50%オフの「Founding Member」価格

### 1.2 THE TOLLの独自価値と支払意欲の推定

THE TOLLのWTP（Willingness to Pay）を他のカテゴリと比較する。

**2つの代替コストで価値を定義できる:**

1. **サイトブロッカーとして**: Freedom ($8.99) ≒ BlockSite ($3.99) の中間 → $5-7/月
2. **フィットネスアプリとして**: スポーツジム月会費（平均$30-70）の代替手段 → $10+/月
3. **掛け合わせ効果**: 「ジムに行かずに自宅で運動習慣が身につく」なら $8-12/月

ただし、Chrome拡張という認知的フレーミング（「拡張機能は無料が当たり前」）により、実際の支払意欲は割り引かれる傾向がある。

**推定WTP分布:**
- 価格感応層（学生・フリーランス）: $2-4/月 → この層はFreeプランへ
- メイン層（20-35歳会社員）: $4-8/月 → Proプランのコアターゲット
- ヘビーユーザー（フィットネス意識高い）: $8-15/月 → 追加機能があればPro+も可

**根拠:**
- $4.99は「$5以下」の心理的バリアを越えない（.99効果）
- Freedom ($8.99) の約半額でポジショニングが明快
- 年額$39.99はCold Turkey（買い切り$39）と同額 → 「毎年払う価値がある」比較優位

#### JPY推奨価格

| プラン | 月額 | 年額（月換算） | 年額（一括） |
|---|---|---|---|
| Free | ¥0 | - | - |
| Pro | ¥750/月 | ¥500/月 | ¥5,980/年 |

**根拠:**
- ¥750は「1000円以下」の心理的閾値を下回る
- App Store/Google Play標準価格帯（¥600〜¥1,000）と整合
- 年額¥5,980は「6,000円以下」の心理的バリア回避
- 月換算¥500は「コーヒー1杯分」のアナロジーが使える

**為替レート補正**: 単純な為替換算（$4.99 = ¥750）ではなく、日本市場の購買力平価に基づく独立価格設定を推奨。

### 1.4 年額割引率の最適化

**現在の研究知見:**
- 最適年額割引率は月額の**33-40%オフ**（月換算で60-67%に相当）
- 割引が25%未満だとアップセル動機が弱い
- 割引が50%超だと月額の存在価値が薄れる

**推奨割引設定:**
- Pro月額 $4.99 → 年額月換算 $3.33（**33%オフ**）
- 年額一括 $39.99（$59.88 → $39.99、実質33%オフ）

これにより**LTV（年額）は月額継続より確実で、キャッシュフローも安定**する。

## 3. トライアル戦略の最適化

### 3.1 現行7日間トライアルの評価

**課題:**
- 7日間は習慣形成には短すぎる（習慣は平均21-66日で定着）
- 「本当に効くのか」を体感するには2週間以上必要
- 週末を2回経験できない（週末のSNS誘惑が最大の試練）

**評価: C（改善余地大）**

### 3.2 代替案の比較分析

| 戦略 | 期間/設計 | CVRへの期待効果 | 不正利用リスク | 推奨度 |
|---|---|---|---|---|
| 現行（7日フル） | 7日・全機能 | 中 | 低 | C |
| 14日フル | 14日・全機能 | 高 | 低 | **A** |
| 30日フル | 30日・全機能 | 中（長すぎて忘れる） | 中 | B |
| 段階的制限 | 日々機能が減る | 高（損失回避を刺激） | 低 | **A** |
| Feature-limited | 一部機能のみ | 低（価値が伝わりにくい） | 低 | C |
| Reverse Trial | 全機能→徐々に制限 | 最高（サンクコスト刺激） | 低 | **S** |

### 3.3 推奨トライアル設計: Reverse Trial（段階的制限）

**設計案:**

```
Day 1-14:   [FULL PRO] 全機能フル開放
Day 15-21:  [PRO LITE] カスタムドメイン不可・スケジュール制限
Day 22-28:  [ENHANCED FREE] ブロックサイト5件まで、運動回数制限復活
Day 29+:    [FREE] 完全Free制限に戻る
```

**行動経済学的根拠:**
- **損失回避（Loss Aversion)**: プロスペクト理論によれば、損失の痛みは利得の喜びの約2.25倍。「機能が失われる」痛みは「機能が追加される」喜びより強力
- **サンクコスト効果**: 14日間使い続けたユーザーは「今更やめたくない」心理が働く
- **保有効果（Endowment Effect）**: 一度手に入れた機能（全機能アクセス）を手放したくない

**通知設計（機能制限のお知らせ）:**
- Day 12: 「トライアル終了まで3日。〇〇回の運動達成！この記録を守るには→」
- Day 14: 「明日からカスタムサイトブロックが制限されます。今すぐPro継続→」
- Day 21: 「詳細スケジュール設定が制限されます。24時間以内に決断を→」

### 3.4 トライアル終了時のコンバージョン施策

#### ウィンバックメールシーケンス

| タイミング | 件名 | 内容 | CTA |
|---|---|---|---|
| Day 14（制限開始当日） | 「あなたの記録を守るために」 | 累計運動回数・ブロック成功回数の表示 | 「Proを続ける」 |
| Day 17（3日後） | 「機能が制限されました。でも...」 | 失われた機能の一覧 + 限定割引(20%オフ) | 「割引で復活」 |
| Day 21 | 「最後のオファー」 | 30%オフ・72時間限定 | 「今すぐ申し込む」 |
| Day 30（1週間後） | 「あの頃の習慣、取り戻しませんか？」 | 運動ストリーク復活を促す | 「再開する」 |

#### 期間限定割引の設計

- Day 14-17: 20%オフ（通常価格に戻る前の「早期決断特典」）
- Day 17-21: 30%オフ（「最後のチャンス」感）
- Day 21以降は通常価格（割引乱発で価値が毀損しないよう管理）

## 4. 損失回避・サンクコスト活用

### 4.1 トライアル中の「投資」の可視化

**ダッシュボードで表示すべき累計指標:**

```
┌──────────────────────────────────────────────────────────┐
│  あなたの14日間の実績                                      │
│                                                          │
│  🏋️ 総運動回数      487回                                │
│  🔒 ブロック成功    62回                                  │
│  ⏱️ 節約した時間    3時間24分                             │
│  📅 連続記録         14日（最長記録！）                   │
│                                                          │
│  この成果をProプランで守り続けますか？                     │
│  [Proを続ける - $4.99/月]                                │
└──────────────────────────────────────────────────────────┘
```

**数値化の根拠**: 抽象的な「習慣がついた」より「487回のスクワット」の方が、サンクコストとして機能する。

### 4.2 「この成果を失いますか？」メッセージング設計

**効果的なコピーの原則:**
- 損失フレーミング: 「今やめると〇〇回のデータが消えます」
- 具体的な未来の痛み: 「Free版に戻ると、カスタムサイトブロックが使えなくなります」
- 実績の人格化: 「14日間連続で運動したあなたは、もう上位5%のユーザーです」

**A/Bテスト推奨コピー:**
- バリアントA（損失）: 「この14日間の記録を失いますか？」
- バリアントB（アイデンティティ）: 「運動習慣を身につけた人として、続けますか？」
- バリアントC（社会的証明）: 「同じ目標を持つ23,000人のProユーザーと一緒に続けますか？」

### 4.3 ストリーク・連続記録の活用

**ストリーク設計:**
- 毎日ログイン/運動で「連続記録」を加算
- 記録が途切れるとバッジ消滅（損失回避）
- Proユーザーのみ「ストリーク復活券」（月1回、30日継続で付与）

**通知設計:**
- 「6日連続！明日も続けてストリーク7日達成→特別バッジ獲得」
- 「昨日の記録が途切れました。今日再開してストリークを守ろう」

### 5.1 ローンチ価格（Founding Member）

**設計:**
```
期間: リリース後最初30日間
価格: 通常の50%オフ
  - Pro月額: $2.49/月（通常$4.99）→ ¥380/月（通常¥750）
  - Pro年額: $19.99（通常$39.99）→ ¥2,980（通常¥5,980）
バッジ: 「Founding Member」プロフィールバッジ（永続付与）
上限: 先着500名
```

**行動経済学的根拠:**
- **希少性**: 「先着500名」で行動を促進
- **コミットメント**: 初期コストを低くすることで、継続課金の初期障壁を下げる
- **アイデンティティ**: 「Founding Member」バッジは特別感とコミュニティへの所属意識を付与

### 5.2 紹介プログラム

**設計:**
```
紹介した側:   登録1人につき+7日間無料延長（最大3ヶ月）
紹介された側: トライアル7日→14日に延長
```

**上限の設定:** 紹介は月3人まで（スパム防止）。21日間の延長が最大報酬。

**行動経済学的根拠:**
- **互恵性**: 「友達のために特典を取ってあげた」感覚
- **社会的証明**: 友人の紹介はCVRが通常の5-10倍高い

## 6. THE TOLL固有の「運動×課金」掛け合わせ戦略

### 6.1 運動で割引を獲得する仕組み

**「Exercise Reward」プログラム設計:**

```
月間運動マイルストーン:
  100回達成  → 翌月10%割引
  300回達成  → 翌月15%割引
  500回達成  → 翌月20%割引（最大）

累計運動マイルストーン（永続割引）:
  通算1,000回 → $0.50/月割引
  通算5,000回 → $1.00/月割引
  通算10,000回 → 「Legend」バッジ + 20%永続割引
```

**行動経済学的根拠:**
- **目標勾配効果（Goal Gradient Effect）**: ゴールに近づくほどモチベーションが上がる
- **変動報酬**: 達成時のサプライズ（「今月も頑張った！割引獲得！」）でドーパミン分泌
- **進捗の可視化**: ゲーミフィケーションと課金の融合でチャーン率低下

**重要な制約**: 割引は最大20%まで。プロダクトが成立する利益率を確保する。

### 6.2 ゲーミフィケーション要素と課金の連動

**Pro限定ゲーミフィケーション:**

| 機能 | Free | Pro |
|---|---|---|
| 週間/月間ランキング | - | ✓ |
| バッジコレクション | 3種 | 30種以上 |
| カスタムチャレンジ | - | ✓ |
| 友人とのチャレンジ | - | ✓ |
| 運動統計・グラフ | 3日分 | 全期間 |
| ストリーク復活券 | - | 月1枚 |
| カスタムロック画面メッセージ | - | ✓ |

**Pro限定機能の「チラ見せ」戦略:**
- Freeユーザーにランキング画面を表示するが、自分の順位にブラーをかける
- 「Proにアップグレードして順位を確認→」のCTAを配置

### 6.3 コミュニティ要素のPro限定化

**コミュニティ設計案（将来ロードマップ）:**

```
Phase 1（MVP後）:
  - 月間チャレンジ（Pro限定参加）
  - 匿名ランキング（Pro）

Phase 2（v1.2+）:
  - 友達招待・プライベートグループ（Pro）
```

**ネットワーク効果の活用:**
- コミュニティが育つほど、Proの価値が増す（コンテンツ/競争相手が増える）
- Freeユーザーも「見える」形で参加させ、Proの「にぎわい」を体感させる

## 8. 行動経済学的原理との紐付き まとめ

| 施策 | 活用原理 | 期待CVR効果 |
|---|---|---|
| Reverse Trial（段階的制限） | 損失回避、保有効果 | +30-50% |
| 累計運動データの可視化 | サンクコスト、コミットメント | +20-30% |
| 年額33%オフ | 将来割引（Hyperbolic Discounting）の逆用 | +15-25% |
| Founding Member限定感 | 希少性、FOMO | +40-60%（初期） |
| 3プラン構成のデコイ | 非対称優位性、選択の簡単化 | +10-20% |
| ストリークと損失 | 損失回避、コミットメント一貫性 | +15-25% |
| 紹介プログラム | 互恵性、社会的証明 | +30-50%（紹介経由） |
| 運動で割引 | 目標勾配、変動報酬 | チャーン-20% |
